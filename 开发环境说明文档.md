# 开发环境说明文档

## 1. 开发环境概述

本文档用于说明 CampusHub（校园互助服务平台）项目的开发环境配置要求，确保团队成员使用一致的环境进行开发与调试。

## 2. 基础环境要求

### 操作系统
- Windows
- macOS
- Linux

### 版本控制
- Git+GitLab

### IDE / 编辑器（推荐）
IntelliJ IDEA

## 3. 前端开发环境

| 工具 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | 18.x 或 20.x | 推荐使用 LTS 版本 |
| npm | 9.x 或 10.x | 包管理工具 |
| Vue CLI / Vite | 项目内置 | 构建工具 |
| 浏览器 | Chrome / Edge 最新版 | 调试推荐 |

### 前端依赖（关键）
- Vue 3.x
- Vue Router
- Element Plus
- Axios

### 前端启动命令
```bash
npm run dev:frontend
```
****
**访问前端** http://localhost:5173
## 4. 后端开发环境

| 工具 | 版本要求 | 说明 |
|------|----------|------|
| JDK | Java 17 | 必须使用 Java 17 |
| Maven | 3.8+ | 项目构建与依赖管理 |
| MySQL | 8.0+ | 数据库 |
| Postman | 最新版 | 接口测试（可选） |

### 后端依赖（关键）
- Spring Boot 3.x
- MyBatis
- Spring Security
- MySQL Connector

### 后端启动命令
```bash
npm run dev:backend
```
**访问后端测试** http://localhost:8080/hello


## 5. 数据库环境

* **数据库类型**：MySQL
* **字符集**：utf8mb4


## 6. 其他工具与环境

| 工具 | 用途 |
|------|------|
| Git | 版本管理 |
| GitLab CI | 自动化测试 |
| Postman | 接口调试与文档 |
| Maven | 后端依赖管理 |

## 7. 常见问题与注意事项

### 后端
- 更新 `pom.xml` 后，请使用 Maven 刷新依赖
- 确保 JDK 版本为 17，否则可能运行失败
- 检查 `application.properties` 中的数据库连接配置

### 前端
- 确保 Node.js 版本 ≥ 18
- 若端口被占用，可在 `vite.config.js` 中修改端口

## 8. 环境验证方式

### 前端
访问 http://localhost:5173， 能看到页面即成功。

### 后端
取消后端安全依赖，然后访问 http://localhost:8080/hello ，页面应该显示`Hello，Spring Boot！`

### 数据库
通过 MySQL 客户端连接，能正常查询表结构。
